Install

For Window:
MolAICal need not be installed, just decompress the MolAICal file. MolAICal is command line tool. You can use the absolute path of molaical on DOS or powershell of windows. For example: if MolAICal is placed on the D: disk, then you can run as blow:

PS D:\> D:\MolAICal-win64\molaical.exe

Press Enter Key, it should show message like: The parameter can not be found, please input the right parameter.

For Linux:
1. tar -xzvf MolAICal-linux64-xxx.tar.gz
Notice: Please replace the corrected characters in MolAICal-linux64-xxx.tar.gz according to your downloaded MolAICal.
2. cd MolAICal-linux64-xxx
3. chmod +x install.sh
4. ./install.sh

To use MolAICal expediently, users can set system path of MolAICal. So the command molaical.exe can be used without given path.

Notice: due to the system compatibility problem in Linux operating system, the “AIGenMols” of MolAICal may not run in your system, you can package your trained model by PyInstaller (see the first example in “Specialized topics of MolAICal” of MolAICal tutorials), or choose win64 version of MolAICal to generate your desired number of molecules because the “AIGenMols” of MolAICal have good compatibility in Windows operating system. The memory of 32-bit MolAICal (win32) is limited up to ~2GB. The memory of 64-bit MolAICal (win64, linux64) is unlimited. We recommend 64-bit version of MolAICal.
